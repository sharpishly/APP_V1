\item {\bfseries{Client ID}} and {\bfseries{Client Secret}} from your Linked\+In app.

\item Access to Linked\+In API permissions (e.\+g., {\ttfamily r\+\_\+liteprofile}, {\ttfamily r\+\_\+emailaddress}, {\ttfamily w\+\_\+member\+\_\+social}). Note that job-\/related APIs may require Talent Solutions partner access.

\hypertarget{md_dev_8sharpishly_8com_2website_2vendor_2LinkedIn_2README_autotoc_md97}{}\doxysection{\texorpdfstring{Important Notes}{Important Notes}}\label{md_dev_8sharpishly_8com_2website_2vendor_2LinkedIn_2README_autotoc_md97}

\begin{DoxyItemize}
\item {\bfseries{API Permissions}}\+: Job-\/related APIs (e.\+g., job search, applications) often require Linked\+In Talent Solutions partner access. Without it, you may encounter {\ttfamily 403 Forbidden} errors. Apply for access via the \href{https://developer.linkedin.com}{\texttt{ Linked\+In Developer Portal}}.
\item {\bfseries{User Registration}}\+: Linked\+In does not allow direct user creation via the API. Users must register on Linked\+In\textquotesingle{}s website, and the API handles authentication.
\item {\bfseries{Security}}\+:
\begin{DoxyItemize}
\item Store the access token securely (e.\+g., in a database or session).
\item Avoid exposing {\ttfamily Client ID} and {\ttfamily Client Secret} in public code.
\item Use HTTPS for all requests to prevent token leakage.
\end{DoxyItemize}
\item {\bfseries{Rate Limits}}\+: Linked\+In imposes API rate limits. Monitor your app\textquotesingle{}s usage in the Developer Portal.
\item {\bfseries{Error Handling}}\+: The class includes basic error handling. Enhance it based on your application\textquotesingle{}s needs (e.\+g., logging, user-\/friendly messages).
\end{DoxyItemize}\hypertarget{md_dev_8sharpishly_8com_2website_2vendor_2LinkedIn_2README_autotoc_md98}{}\doxysection{\texorpdfstring{Troubleshooting}{Troubleshooting}}\label{md_dev_8sharpishly_8com_2website_2vendor_2LinkedIn_2README_autotoc_md98}

\begin{DoxyItemize}
\item {\bfseries{Authentication Errors}}\+: Ensure your {\ttfamily Client ID}, {\ttfamily Client Secret}, and {\ttfamily Redirect URI} match those in the Linked\+In Developer Portal.
\item {\bfseries{Permission Denied}}\+: If job-\/related endpoints return errors, verify your app has the necessary permissions or apply for Talent Solutions access.
\item {\bfseries{c\+URL Issues}}\+: Ensure the c\+URL extension is enabled in your PHP environment ({\ttfamily php\+\_\+curl}).
\end{DoxyItemize}\hypertarget{md_dev_8sharpishly_8com_2website_2vendor_2LinkedIn_2README_autotoc_md99}{}\doxysection{\texorpdfstring{Limitations}{Limitations}}\label{md_dev_8sharpishly_8com_2website_2vendor_2LinkedIn_2README_autotoc_md99}

\begin{DoxyItemize}
\item {\bfseries{Job APIs}}\+: Public access to job postings and applications is restricted. You may need partner status for full functionality.
\item {\bfseries{User Registration}}\+: Direct user creation is not supported. Rely on Linked\+In\textquotesingle{}s OAuth flow for authentication.
\item {\bfseries{API Changes}}\+: Linked\+In\textquotesingle{}s API may change. Check the \href{https://docs.microsoft.com/en-us/linkedin/}{\texttt{ Linked\+In API documentation}} for updates.
