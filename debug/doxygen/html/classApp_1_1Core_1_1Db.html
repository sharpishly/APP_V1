◆ joins_many() [1/2]

App\Core\Db::joins_many (   $data,
  $conditions 
)

Joins multiple tables based on the 'joins_many' key in the $conditions array.

Parameters
array$dataThe current data array containing the SQL query.
array$conditionsAn array containing join conditions. The 'joins_many' key should contain an array of join definitions, where each definition is an associative array with:
  • 'type': (string) The type of JOIN (e.g., 'INNER', 'LEFT', 'RIGHT').
  • 'table': (string) The table to join.
  • 'on': (array) An associative array defining the join condition:
  • 'this_table_col': (string) The column from the main table.
  • 'other_table_col': (string) The column from the joined table.
Returns
array The updated data array with the constructed join clauses appended to the SQL query.
